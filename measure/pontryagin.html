39.6 A few harder problems to think about

Problem 39A.

If G is compact, so G^ is discrete, describe the dual measure ŒΩ. You can read it off Theorem¬†39.3.1. It is the counting measure.

Problem 39B.

Show that an LCA group G is compact if and only if G^ is discrete. (You will need the compact-open topology for this.) After Pontryagin duality, we need to show G compact implies G^ discrete and G discrete implies G^ compact. Both do not need anything fancy: they are topological facts.
