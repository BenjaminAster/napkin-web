Chapter 48 Morphisms between Riemann surfaces

48.1 Definition

The definition is what we would expect — since a Riemann surface’s main feature is a complex structure, a map f:ℂ→ℂ is a morphism between Riemann surfaces if and only if it is holomorphic.

Definition 48.1.1.

Let X and Y be Riemann surfaces. A mapping f:X→Y is holomorphic at p∈X if and only if there exists charts ϕ1:U1→E1 on X with p∈U1 and ϕ2:U2→E2 on Y with f⁢(p)∈U2 such that the composition ϕ2∘f∘ϕ1−1 is holomorphic at ϕ1⁢(p). We say f is a morphism between Riemann surfaces if and only if it is holomorphic at all points of X.

In other words: f is holomorphic if and only if it is holomorphic as function mapping between local coordinates.

Example 48.1.2

Some examples follows.

  • •

    The function f:ℂ→ℂ by f⁢(x)=x3 is a morphism.

    Note that this function is not bijective. At each point p≠0, there is an open neighborhood on which f has an inverse, but f has no inverse at 0.

  • •

    The embedding of the complex plane into the Riemann sphere, ℂ↪ℂ∞, is a morphism.

48.2 Functions to the Riemann sphere

Prototypical example for this section: The meromorphic function 1z can be made into a holomorphic ℂ→ℂ∞ function.

In this section, we will see that the Riemann sphere ℂ∞ can be viewed as “ℂ with a point at infinity added”. This interpretation allows us to interpret meromorphic functions f:X→ℂ as holomorphic maps g:X→ℂ∞, which allows a much better handling of meromorphic functions — there’s no longer any singularity, the resulting function g is holomorphic everywhere!

First, we see that ℂ∞ can be naturally interpreted as ℂ with a single point added: With notation as in Example 47.4.2, identify ℂ∞∖{N} with E1 (and thus with ℂ) through the map ϕ1, and we let ∞ be the point N.

Question 48.2.1

Convince yourself that it makes sense to call the point ∞ — for every sequence of points {zi} on ℂ such that |zi|→+∞, then ϕ1−1⁢(zi)→∞ on ℂ∞ as a topological space.

So, let X be a Riemann surface, and f:X→ℂ be a meromorphic function on X. Naturally, g can be defined by

g⁢(z)={f⁢(z)if ⁢f⁢(z)≠∞∞if ⁢f⁢(z)=∞.

Then g is continuous — but furthermore, it’s analytic.

Question 48.2.2

Clearly, at points z∈X where g⁢(z)≠∞, then g is analytic.

Convince yourself that g is also analytic at z∈X where g⁢(z)=∞. (With notation as in Example 47.4.2, take a small open set U⊆X, and re-parametrize gimg⁢(U)⊆ℂ∞ by t=1/z.)

Therefore,

Proposition 48.2.3

There is a one-to-one correspondence between meromorphic functions f:X→ℂ and holomorphic maps g:X→ℂ∞ such that g is not identically ∞.

Or, more informally,

Moral 48.2.4 —

Plugging in the hole at ∞ of ℂ allows us to analytically extend meromorphic functions to ℂ∪∞ maps which is holomorphic everywhere.

48.3 Some other nice properties

We have just seen in the last section that the Riemann sphere ℂ∞ allows us to remove the singularities of a meromorphic functions.

Informally speaking, this is because ℂ∞ is a “compactfication” of ℂ — adding a point to make it compact — and compact Riemann surfaces enjoy many nice properties.

Proposition 48.3.1

Let X and Y be compact, f:X→Y be holomorphic and not constant. For each point y∈Y, define dy be the total multiplicity of the points in the preimage of y.

Then, dy is well-defined and constant.

You can see why this proposition is surprising:

Example 48.3.2 (The proposition does not hold for smooth compact manifolds)

Consider the following function f:X→Y between compact smooth real 1-manifold, depicted as a plot with x and y-axis. (Note that a compact 1-manifold cannot be embedded into ℝ, because compact subsets of ℝ are closed and bounded, thus necessarily have a boundary. A proper graph would live in a 4-dimensional space, which is rather difficult to visualize, so we settle with an approximate representation.)

Here, X and Y are both isomorphic to the unit circle.

We count the number of points in the fiber above each point in Y:

  • •

    Above point A, there are infinitely many points.

  • •

    Above point B, there is only one point. (You can argue that this point has “multiplicity 2” however)

  • •

    Above point C, there are two points.

  • •

    Above point D, the fiber is empty.

Definition 48.3.3.

The value dy above is called the degree of the map f, written deg⁡(f).

Example 48.3.4

The map z↦zk, when extended to a ℂ∞→ℂ∞ map, has degree k.

If t≠0, then we know that t has k distinct k-th roots. But if t=0, its preimage only consist of the point 0 — in this case, we wish to say z=0 is a “multiple point” — we will formalize it next section when we defines the multiplicity of a map.

If you have read Section 74.1, this is in fact the same as the concept of a degree in homology when X and Y are both Riemann spheres — it counts how many spherical bags that im⁡f consists of. But, in this case, the theory is extra nice — not only that the graph is homotopy equivalent to one that covers each point d times, but each point is in fact covered exactly d times!

This theme will be recurrent in complex analysis and Riemann surfaces. Basically:

Moral 48.3.5 —

If the “things” are counted properly, the formula is very nice.

The proof of the proposition is not difficult — the main observation is that the theorem is true for functions of the form f⁢(z)=zn, and locally around each point p∈X, f is either an isomorphism or has the form above. So dy is locally constant, and thus constant because Y is connected.

48.4 Multiplicity of a map

Prototypical example for this section: f⁢(x)=(x−3)2 has mult3⁡(f)=2.

In the previous section, we informally talk about the multiplicity of a map at a point. We will rigorously define it in this section.

Example 48.4.1

Consider the map f:ℂ→ℂ given by f⁢(z)=z5+1.

Above each point y∈ℂ, the fiber fpre⁢(y) has 5 points — except when y=1, then fpre⁢(1)={0} has only 1 point.

This behavior is undesirable, and we would like to say that the function f maps 5 “identical copies” of the point 0 to the point 1. (Another way you could see it is that, for each sequences {yi} converging to 1, there are 5 different sequences {xi} converging to 0 such that f⁢(xi)=yi for each i.)

Inspired by this, we will define multiplicity in a way such that:

  • •

    z↦zm has multiplicity m, for integer m≥1.

  • •

    If we perform an analytic reparametrization of the source or the target, then the degree does not change.

Turns out these two properties completely defines the degree! We have the following.

Proposition 48.4.2

Let f:X→Y be a nonconstant holomorphic map defined at p∈X. Then there is a unique integer m≥1 such that, for every chart ϕ2:U2→V2 on Y centered at f⁢(p) (that is, ϕ2⁢(f⁢(p))=0), there is a chart ϕ1:U1→V1 on X centered at p such that the induced map ϕ2∘F∘ϕ1−1:V1→V2 has the form z↦zm.

In other words, once we fix a chart of Y, there exist a chart of (an open subset of) X such that the induced map between open subsets of ℂ is a power map; furthermore, the exponent is independent of the selection.

Moral 48.4.3 —

Every map looks locally like z↦zm.

Proof.

Essentially, use the Taylor expansion to determine m, then the selection of ϕ1 is pretty much fixed by the restrictions. ◻︎

Definition 48.4.4.

The value m above is the multiplicity of f at point p, written multp⁡(f).

Example 48.4.5 (More examples of multiplicity of a map at a point)

We consider some examples.

  • •

    The function z↦z−2, extended to a ℂ→ℂ∞ map, has multiplicity 2 at point 0 — “two copies” of the point 0 is mapped to the point ∞.

  • •

    The function f⁢(z)=(z−1)⁢(z−2)5 has mult2⁡(f)=5 — more generally, if p is a root of f, then multp⁡(f) is the multiplicity of the root.

  • •

    The function z↦z+1 has multiplicity 1 everywhere — in fact, the multiplicity of a nonconstant map at “most” points will be 1.

These are the official terms:

Definition 48.4.6.

A point p such that multp⁡(f)>1 is called a ramification point. In that case, the point f⁢(p) is called a branch point.

48.5 The sum of the orders of a meromorphic function

Yet another case where we get a nice formula.

Example 48.5.1

Let us consider some meromorphic ℂ∞→ℂ∞ functions (defined by extending a ℂ→ℂ function the obvious way), and list the zeros and poles of it (with multiplicity).

Function Zeros Poles
5 None None
(x+1)2 −1, −1 ∞, ∞
1x2+1 ∞, ∞ i, −i
x+1x+2 −1 −2

Every time, the number of zeros equals the number of poles. This is not a coincidence!

Proposition 48.5.2

Let f:X→ℂ be a nonconstant meromorphic function on a compact Riemann surface X. Then

∑pordp⁡(f)=0.

Of course, we need X to be compact — there certainly are ℂ→ℂ functions that has several zeros, but no poles.

Proof.

We extend f to a X→ℂ∞ function, then the sum of multiplicities of points in the fiber of 0 is equal to that in the fiber of ∞. ◻︎

48.7 The identity theorem

The following propositions are expected — the same behavior is seen in complex analysis with holomorphic functions.

Theorem 48.7.1

Let f,g:X→Y be holomorphic maps between Riemann surfaces. If f=g on a nonempty open subset of X, then f=g.

This is the analog of 31C. Note that here the assumption that X is connected is used — the disjoint union of two copies of ℂ is a smooth 2-manifold, but not a Riemann manifold.

That is,

Moral 48.7.2 —

Holomorphic maps are rigid — the value of a function on a tiny subset determines its value everywhere.
